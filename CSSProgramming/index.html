Данная таблица стилей создавалась мной собственноручно, чтобы иметь возможность, при необходимости, выделить соответствующим образом в тексте HTML-документа, в котором описывается программа на языке программирования, имена классов и интерфейсов, полей классов, имена существующих объектов в программе, методов и т.д.

Кроме того, таблица стилей позволяет выделять особым образом блоки кода, делая отступы, выравнивание, а также синтакситечкую подсветку кода (ключевые слова, статические методы, и т.д).

Имена переменных, классов, интерфейсов, описываемые в тексте

Ключевые элементы программы, - имена описываемых объектов, имена имена классов, интерфейсов, методов и т.п. в тексте HTML документа выделяются с помощью HTML тега <span> c классом code. Например:

<span class=code>имя переменной</span>

Такой элемент в документе будет выделен с помощью таблицы стиля и будет отображатся как текст с использованием шрифта Consolas (или Monospace если такой шрифт не будет найден в системе ), размером 13 пунктов, синего цвета.

Блоки с кодом

Блоки кода размечаются следующим образом:

Весь блок с кодом выделяется с помощью тега <div> с классом code. Например:

<div class=code>
   это блок кода
</div>

Данный тег позволяет сместить весь блок c кодом относительно документа, на 2em вправо и на 1.5em вниз и вверх.

Аннотации

Для аннотаций используется тег <div> с классом annotation, например:

<div class=annotation>@Override</div>

В результате, данный элемент программы (@Override) в документе будет выделен серым цветом.

Ключевые слова в программе

Ключевые слова в программе (в блоке кода) размечаются используя тег <span> с классом keyword, например:

<span class=keyword>public void</span>

В результате, отмеченное соответсвующим тегом, ключевое слово в документе будет выделено светло-синим цветом.

Разметка класса в блоке кода

Когда мы описываем класс в блоке кода, - данный элемент HTML страницы будет контейнером <div>. Внутри этого контейнера будет размечено соответсвующей разметкой тело класса.

Имя класса

Для имени класса в таблице стилей никаких правил нет. Поэтому в этом разделе будут рекомендации, а не требования, которых необходимо придерживаться. Эти рекомендации позволяют облегчить верстку и позволяют лучше ориентироваться в коде HTML документа.

Как говорилось ранее, класс в блоке кода это контейнер <div>, и я рекомендую размечать имя класса сразу же после открытия этого тега, используя при этом, если необходимо, тег <span class=keyword>public class</span>, чтобы разметить ключевые слова в программе.

Я рекомендую открывыющую фигурную скобку у класса оставлять на той же строке, в которой вы размечали имя класса, а закрывающую фигурную скобку ставить непосредственно перез закрытием контейнера <div> для блока с классом.

На примере это будет понятнее

<div><span class=keyword>public class</span> MyClassName {
}
</div>


Тело класса

Тело класса, это так же, отдельный, <div> контейнер, в котором используется класс classbody. Этот контейнер начинается после открывающей фигурной скобкой класса и заканчивается перед закрывающей фигурной скобкой.

Данный контейнер позволяет сместить содержимое тела класса вправо на 2em, а также смещает содержимое тела класса вниз на 1em, таким образом производится смещение в тексте прораммы для описания классов.

Пример:

<div><span class=keyword>public class</span> MyClassName {
    <div class=classbody>
      Тело класса
    </div>
}
</div>

Разметка метода в блоке кода

Когда мы описываем метод в блоке кода, - данный элемент HTML страницы будет контейнером <div>. Внутри этого контейнера будет размечено соответсвующей разметкой тело метода.

Имя метода

Для имени метода в таблице стилей никаких правил нет. Поэтому в этом разделе будут рекомендации, а не требования, которых необходимо придерживаться. Эти рекомендации позволяют облегчить верстку и позволяют лучше ориентироваться в коде HTML документа.

Как говорилось выше, метод в блоке кода это контейнер <div>, и я рекомендую размечать имя метода сразу же после открытия этого тега, используя при этом, если необходимо, тег <span class=keyword>public void</span>, чтобы разметить ключевые слова в программе.

Я рекомендую открывыющую фигурную скобку у метода оставлять на той же строке, в которой вы размечали имя метода, а закрывающую фигурную скобку ставить непосредственно перез закрытием контейнера <div> для блока с методом.

На примере это будет понятнее

<div><span class=keyword>public void</span> MyMethod() {
}
</div>


Тело метода

Тело метода, это так же, отдельный, <div> контейнер, в котором используется класс methodbody. Этот контейнер начинается после открывающей фигурной скобкой метода и заканчивается перед закрывающей фигурной скобкой.

Данный контейнер позволяет сместить содержимое тела метода вправо на 2em, а также смещает содержимое тела метода вниз на 0.5em, таким образом производится смещение в тексте прораммы для описания методов.

Пример:

<div><span class=keyword>public void</span> MyMethod() {
    <div class=methodbody>
      Тело метода
    </div>
}
</div>

final переменные (константы)

final переменные в коде программы (константы) можно выделить с помощью тега <span> с классом finalvar. Например:

<span class=finalvar>DEFAULT_LOCATION_POINT</span>

Данный тег позволяет выделить такую переменную в блоке кода в HTML-документе полужирным

Статические методы

Статические переменные в коде программы можно выделить с помощью тега <span> с классом staticmethod. Например:

<span class=staticmethod>assertEquals</span>()

Данный тег позволяет выделить такую переменную в блоке кода в HTML-документе курсивом

Стоки

Строки в коде программы выделяются с помощью тега <span> с классом string. Например

System.out.println(<span class=string>"Hello World!"</span>)

Указанная разметка позволяет выделить строки в блоке кода коричневым цветом.


Перенос на новую строку

Бывают ситуации в программах, когда в одну строку на экране все не умещается и нужно выполнить перенос строк, и при этом, для новой, перенесенной строки, устанавлиаются отступы. Такое встречается в программах чаще всего, когда используются длинные строки, и к ним конкатинируются значения переменных, и т.п. Поэтому я разместил информацию о правилах верстки и использования моей таблицы стилей в разделе описывающем строки.

При возникновении подобной ситуации, и когда необходимо сдвинуть новую перенесенную строку вправо, используется HTML-элемент с классом stringnewline. HTML-элемет может быть любым. Но я рекомендую заключать новую перенесенную строку в тег <div>, применяя у него класс stringnewline, потому что тег <div> блочного типа, и используя его нет необходимости применять тег <br>.

Пример:

System.out.println(<span class=string>"Hello, my name is "</span> +
<div class=stringnewline>myname);</div>
Использование HTML элемента с классом stringnewline сдвигает выделенную строку вправо на 5em.

Комментарии

Для того, чтобы выделить текст программы в блоке кода как комментарий, необходимо использовать элемент HTML с классом comment.

При этом выделенный текст будет отображаться, как комментарий, зеленым цветом. Применяя класс comment можно использовать любой тег. Но рекомендуется использовать, в зависимости от вида комментария теги <span> для комментариев вида // и <div> для комментариев вида /* */

Примеры:

Linen linen = spy(new Linen());  <span class="comment">// Требуется библиотека org.mockito.Mockito.*</span>

<div class=comment>/**<br>
* @param weight<br>
* the weight to set<br>
*/
</div>
public void setWeight(int weight) {
...
}

Тело цикла

Для того чтобы выполнить отступы вправо для кода в цикле (for, while) необходимо код тела цикла поместить в тег-контейнер <div> с классом loopbody.

Тело цикла начинается сразу после открывающейся фигурной скобки цикла (или воображаемой скобки, если её нет в тексте программы) и заканчивается непосредственно перед закрывающейся фигурной скобкой цикла (или вооброжаемой скобкой, если её нет в программе).

Например:

for (AbstractLayer layer : linen.getLayers()) {
<div class="loopbody">
Тело цикла
...
</div>
}

При использовании элемента HTML <div> с классом loopbody выделенный блок сдвигается вправо на 2em, а также опускается на 0.5em.

Тело if

Для того чтобы выполнить отступы вправо для кода в условном операторе if необходимо код в блоках if (или else) поместить в тег-контейнер <div> с классом ifbody.

Тело условного опрератора if (или else) начинается сразу после открывающейся фигурной скобки (или воображаемой, если её нет в тексте программы) и заканчивается непосредственно перед закрывающейся фигурной скобкой (или вооброжаемой скобкой, если в программе нет её ).

Например:

if (a > b)
<div class="ifbody">
System.out.println("a > b");
</div>
else
<div class="ifbody">
System.out.println("b > a ");
</div>

При использовании элемента HTML <div> с классом ifbody выделенный блок сдвигается вправо на 2em, а также опускается на 0.5em.

Тело try-catch-finally

Для того чтобы выполнить отступы вправо для кода в блоках кода try, catch или finally необходимо код в этих блоках поместить в тег-контейнер <div> с классом trycatchbody.

Тело блока кода try, catch или finally начинается сразу после открывающейся фигурной скобки (или воображаемой, если её нет в тексте программы) и заканчивается непосредственно перед закрывающейся фигурной скобкой (или вооброжаемой скобкой, если в программе нет её ).

Например:

try {
<div class="trycatchbody">
Thread.speep(500);
</div>
} catch (InterruptedException ex) {
<div class="trycatchbody">
ex.printStackTrace();
</div>
}

При использовании элемента HTML <div> с классом trycatchbody выделенный блок сдвигается вправо на 2em, а также опускается на 0.5em.
